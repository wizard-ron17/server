Account Balance Endpoint
Retrieve account balance for all tokens, or optionally specify an asset ID/namespace to get a specific token balance for the account.
Wallet Address (required)
Asset ID (optional)
Transfers History Endpoint
Fetch transfer history for a specific account with pagination support. Use Page ID to navigate through results and Page Size to control the number of records per page. Results are sequential, so Page ID 0 and Page Size 10 will loead the 10 most recent transfers, but Page ID 1 and Page Size 10 will load the next 10 transfers.
Wallet Address (required)
Page ID (optional, default: 0)
Results per page (optional, default: 10, less than 500)
